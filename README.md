# **Бот для планирования дел**

проект на Python

автор: Разносилина Алина Б05-211

### **Описание**

Этот бот создан для облегчения планирования. Просто запишите ему при помощи простых
команд все свои важные дела, и дальше cможете проверять, что нужно сделать сегодня, 
в течение месяца и вообще. Бот надежно хранит ваши данные в базе и ничего 
не забудет?.

### **Как использовать?**

#### Немного о версиях
* python 3.10
* Все требуемые библиотеки и версии - в *_requirements.txt_*
* Все файлы из директории также необходимы для нормального функционирования


#### Запуск:
* Нажмите Run *_bot_*

#### Что можно делать?:
* Написать /start - начнете взаимодействие с ботом, он поздоровается, занесет 
имя в базу и подскажет, что нужно написать /help
* В /help бот подскажет, что он умеет делать
* С помощью /record_events сможете записать своё событие, формат подскажет бот
* С помощью /today_events, /moonth_events, /all_events сможете посмотреть все 
записанные события, которые запланированы, соответсвенно, на сегодняшний день, 
ближайший месяц и на всё время
